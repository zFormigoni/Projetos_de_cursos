# Anotações de Markdown 

## Edição de texto
---

### Formatação:

> Escreva:	       
1) *Italic*	
2) **Bold**
3) [Link](http://a.com)
4) [Image](http://url/a.png)

> Ou:
1) _Italic_	
2) __Bold__
3) [Link][1]
⋮
[1]: http://b.org
4) ![Image][1]
⋮
[1]: http://url/b.jpg

---
